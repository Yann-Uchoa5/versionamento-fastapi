Explicação das Rotas da API

Abaixo você encontrará uma breve explicação sobre as principais rotas da API do Sistema de Gerenciamento de Negócios. Cada rota tem um propósito específico e está organizada para facilitar o uso no gerenciamento de produtos, clientes e pedidos.

Rota Clientes

A rota de clientes permite a você visualizar todos os clientes cadastrados no sistema. Ela retorna uma lista com informações como nome, e-mail e telefone. Utilizando a rota GET /clientes, você pode obter esses dados. Caso queira adicionar um novo cliente, use a rota POST /clientes, fornecendo as informações necessárias no corpo da requisição.

Acessar

Rota Produtos

A rota de produtos permite a você visualizar todos os produtos cadastrados no sistema. Ela retorna informações como nome do produto, descrição e preço. Você pode obter esses dados utilizando a rota GET /produtos. Para adicionar um novo produto, utilize a rota POST /produtos.

Acessar

Rota Pedidos

A rota de produtos permite a você visualizar todos os pedidos cadastrados no sistema. Ela retorna informações como ID do produto, descrição e status. Você pode obter esses dados utilizando a rota GET /pedidos. Para adicionar um novo produto, utilize a rota POST /produtos.
